Learning the keyboard and alphabet

Например ваш малыш очень любит запускать browser открывать в нем youtube и смотреть в нем мультфильмы
Вам нужно спрятать иконку browser с рабочего стола
Вам нужно скачать скрипт 
Вам нужно разместить на рабочем столе lnk для запуска powershell скрипта
Вам нужно сделать иконку у lnk, чтобы она была как у спрятанного browser
Малыш стартует иконку любимого браузера и получает окно powershell
Ему нужно ввести с клавиатуры пароль который он видит перед глазами.
Когда он будет вводить пароль он будет озвучиваться
Когда пароль будет введен верно запустится browser на 5 минут
Затем все начинается с начала

- For example, your child loves to run browser to open youtube and watch cartoons
- You need to hide the browser icon from the desktop
- You need to download the script
- You need to place on the desktop lnk to run powershell script _start_exe.ps1
- You need to make an icon lnk to it was like a hidden browser
- The kid starts the icon favorite browser and gets the powershell window
- He needs to key in a password which he sees before his eyes.
- When he enter the password it will be voiced
- When the password is entered correctly starts the browser for 5 minutes
- Then it all starts over again

Для случая если у вас английская версия Windows 
И из-за этого, у вас в системе отсутствует русский голос для проговаривания.
Решение можно найти здесь 
https://social.technet.microsoft.com/Forums/ru-RU/812d7ef6-7476-46cd-b3b6-f68e86fe3986/powershell-string-?forum=scrlangru#812d7ef6-7476-46cd-b3b6-f68e86fe3986

Короткая инструкция того что нужно сделать 
1) содержимое архива ru-RU.rar кладем по адресу  - C:\Windows\Speech_OneCore\Engines\TTS\ru-RU
2) развернуть ветку реестра TTS_MS_RU-RU_IRINA_11.0.reg
